Описание
Цикл лабораторных работ (№1–9) по курсу «Системы искусственного интеллекта» (МТУСИ, направление 09.03.02 «Прикладные информационные системы»). В ходе работ создано веб-приложение на Django – блог с публикацией статей, регистрацией пользователей, интерактивностью на JavaScript/jQuery и параллакс-эффектом.

Требования
Python 2.7 / 3.x

Django 1.4.x (совместимость с методичкой)

Виртуальное окружение (рекомендуется)

Установка и запуск
Клонировать репозиторий.

Перейти в папку нужной лабораторной (например, cd lab3).

Создать и активировать виртуальное окружение.

Установить зависимости: pip install -r requirements.txt.

Выполнить миграции: python manage.py migrate.

Запустить сервер: python manage.py runserver.

Открыть в браузере http://127.0.0.1:8000/.

Структура
Каждая лабораторная находится в отдельной папке (lab1…lab9) и является самостоятельным Django-проектом.

Лабораторные работы (кратко)
Основы Python, установка Django – скрипт на Python, первый проект, админка.

Flatpages и статика – создание страницы с CSS и изображением.

Модели и вывод статей – модель Article, миграции, список всех записей.

Детальная страница и стилизация – страница одной статьи, оформление по макетам.

Форма создания поста – добавление статей через веб-интерфейс с валидацией.

Регистрация и авторизация – создание пользователей, ограничение доступа.

JavaScript – сворачивание/разворачивание статей.

jQuery и анимация – подсветка статей, увеличение логотипа.

Parallax – параллакс-эффект для иконок и логотипа.
